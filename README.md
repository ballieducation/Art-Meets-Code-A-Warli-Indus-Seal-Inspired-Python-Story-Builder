📖 Project Introduction

Art Meets Code: A Warli & Indus Seal Inspired Python Story Builder is an interactive storytelling application built with Python and Pygame.
The project allows learners to create stories by arranging visual cards into meaningful sequences. It combines art, history, and technology by drawing inspiration from Indus Valley seals and Warli tribal art, making narrative planning more engaging and culturally rooted.

Key Features:

🎴 Card-based storytelling: Drag and drop cards to build narratives.

🧑‍🎓 Player Profile: Collects learner’s name, age, and grade.

📜 Story Introduction: Provides background context before gameplay.

🧩 Interactive Sequencing: Encourages logical and lateral thinking.

📝 Quiz & Analytics: Evaluates learning outcomes with scores and reflections.

🎨 Cultural Design: Inspired by traditional Indian art forms.

This project supports competency-based learning aligned with NEP 2020 and NCF 2023, encouraging creativity, problem-solving, and multimodal literacy in classrooms.
